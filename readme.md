## YB-HelloApp

### 1 APP概述

基于apicloud平台，使用了aui/swipper等第三方组件来完善样式。

主要是用于测试自己开发的一些自定义模块及某些官方模块。



##### 需添加自定义模块：

- moduleIflyVoice
- moduleProcessAlive
- pdfReader
- moduleUHF
- moduleNFC
- OtherOffLineSpeakModule
- moduleUHF_I6310
- moduleNFC_2023
- moduleNFC_ZBL

##### 需添加官方模块：

- iflyRecognition
- IFlyVoice   (废弃)
- smsListener
- acc5Util
